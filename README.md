## INSTALLATION

1. Clone the repo
   ```sh
   git clone https://github.com/PiotrSierant/vue-studying.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start project
   ```sh
   npm run serve
   ```
